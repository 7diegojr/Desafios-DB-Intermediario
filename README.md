# Desafio DB Intermediário

Esse desafio é voltado para a prática de operações intermediárias em SQL, trabalhando com consultas mais complexas e manipulação de dados.

## Objetivo do Desafio

O objetivo deste desafio é trabalhar com conceitos intermediários de SQL, como:

- Criação de tabelas e inserção de dados
- Utilização de comandos como `INNER JOIN`, `DELETE`, `CASE`, entre outros
- Elaboração de consultas para recuperar, relacionar e manipular dados de maneira selecionada
